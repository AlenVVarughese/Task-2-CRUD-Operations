# Task-2-CRUD-Operations

## Objective

The objective of this task was to practice basic SQL data manipulation operations on the Library Management System database created in Task 1.

## Tools Used

* MySQL Workbench
* MySQL Server
* GitHub

## Operations Performed

### INSERT

Inserted sample records into:

* Authors
* Categories
* Books
* Members
* Loans

### NULL and DEFAULT Values

Used `NULL` where optional data was unavailable, such as missing email addresses or return dates. Default values can also be used to automatically assign predefined values when no value is provided.

### UPDATE

Updated existing records using the `UPDATE` statement with appropriate `WHERE` conditions.

### DELETE

Deleted selected records using the `DELETE` statement with `WHERE` conditions to avoid removing unintended data.

### SELECT

Verified all changes using `SELECT` queries after each operation.

## SQL Concepts Covered

* INSERT INTO
* SELECT
* UPDATE
* DELETE
* WHERE Clause
* NULL Values
* DEFAULT Values
* Primary Key
* Foreign Key

## Outcome

Successfully performed CRUD (Create, Read, Update, Delete) operations on the Library Management System database while maintaining data integrity through primary and foreign key relationships.
